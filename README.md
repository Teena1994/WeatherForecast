Mongo DB Set Up
=========================================
1. Database Name : MML
2. Collection name : state_details
3. Import data from 'MML.state_details.json' file  to the collection 'state_details'

To Run Project
=========================================
1. Go to app folder
2. npm install
3. node server.js
4. go to client folder
5. ng serve
6. update in .env file mongo uri. by default connected to localhost.
